# Rhyme-lyricist-with-KoGPT2

> KoGPT2 기반의 라임을 고려한 한국어 가사 생성 모델

GPT-2는 주어진 텍스트의 다음 단어를 잘 예측할 수 있도록 학습된 언어모델이며 문장 생성에 최적화 되어 있습니다.<br>
Rhyme-lyricist는 한국어 성능을 끌어올린 KoGPT2를 통해 구현한 라임을 고려한 가사 생성 모델 입니다.
